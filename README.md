# Priyansh Portfolio

A polished personal portfolio website for Priyansh Patel, AI Engineer.

This site presents:

- production AI work such as Chitr, a real-time loss-prevention system
- offline-first software such as the JalaSai Garage System
- experience across applied ML, agentic AI, and data science
- education, skills, and contact details in a recruiter-friendly layout

## Stack

- HTML
- CSS
- Vanilla JavaScript
- GitHub Pages ready

## Run locally

1. Open the folder in a browser or serve it with any static server.
2. Edit `index.html`, `styles/style.css`, and `script.js` to customize content.
3. Push to GitHub Pages to publish.

## Notes

- The site uses no framework and no build step.
- Content is aligned with Priyansh Patel's AI engineering resume.
